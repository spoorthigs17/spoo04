# VetCart Veterinary Store (Flask)

This project is a veterinary e-commerce demo built with Python Flask, HTML, CSS, and JavaScript.

## Folder structure

```
VetCart/
│
├── app.py
│
├── templates/
│   ├── index.html
│   ├── products.html
│   ├── cart.html
│   └── payment.html
│
└── static/
    ├── styles.css
    └── script.js
```

## Run locally

1. Install Flask:

```
pip install flask
```

2. Run the Flask app:

```
python app.py
```

3. Open in browser:

```
http://127.0.0.1:5000
```

## Notes

- This is a demo site and does not include payment processing.
- Use the Flask structure to present your project as a Python-based web application.
